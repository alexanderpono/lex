- это транслятор, у которого объектный код — исполняемый модуль в формате конкретной ОС

1. Компилятор – определение №2

- это транслятор, у которого объектный код — это программа на языке более низкого уровня

1. Транспилятор

- преобразует входную программу на языке высокого уровня в эквивалентную программу на языке высокого уровня (пример: babel)

1. Интерпретатор

= транслятор + блок исполнения только что скомпилированной программы

4. Грамматика математического выражения

<MathExpression> ::= <therm> | <MathExpression> + <therm> <MathExpression> - <therm>

<therm> ::= <factor> | <therm> * <factor> | <therm> / <factor>

<factor> ::= <number> | ( <MathExpression> ) | <factor> ^ <factor>

<number> ::= <ID>


<ID> — идентификатор

( ) + - * / ^ — разделители из таблицы limiters лексического анализатора

| — обозначение альтернативы

4. Простой язык с математическими выражениями

<Program> ::= <callFunction>

<callFunction> ::= <ID> ( ) | <ID> (<parameterList>)

<parameterList> ::= <expression> | <string> [ , <parameterList> ]

<expression> ::= <therm> + <expression> | <therm> - <expression> | <therm>

<therm> ::= <factor> * <therm> | <factor> / <therm> | <factor>

<factor> ::= <number> | ( <expression> ) | <factor> ^ <factor>

<number> ::= <ID>


<ID> — идентификатор

( ) + - * / ^ — разделители из таблицы limiters лексического анализатора

| — обозначение альтернативы
